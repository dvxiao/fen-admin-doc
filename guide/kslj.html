# 项目简介

FEN-ADMIN 基于 Egg、Mysql、Sequelize、Redis、Vue、Element-UI/vue-element-admin 的前后端分离的后台管理系统， 项目采用按功能分模块的开发方式，权限控制采用 RBAC 思想，支持前端菜单动态路由等

# 主要功能

  • [x] 用户管理：提供用户的相关配置，新增用户后，默认密码为123456
  • [x] 角色管理：对权限与菜单进行分配，可根据设置角色的数据权限
  • [x] 菜单管理：已实现菜单动态路由，后端可配置化，支持多级菜单
  • [x] 系统日志：记录用户操作日志与异常日志，方便开发人员定位排错
  • [x] 定时任务：加入任务日志，任务运行情况一目了然
  • [] 七牛云存储：可同步七牛云存储的数据到系统，无需登录七牛云直接操作云数据

    │       ├── Slider (侧边)

    │       └── index.vue
    ├── views (基础页面)
    │   ├── layout (整体布局)
    │   │   ├── process (任务栏)
    │   │   ├── slider (侧边)
    │   │   ├── topbar (顶部)
    │   │   └── index.vue
    │   ├── 403
    │   ├── 404
    │   ├── login (登录)
    │   ├── public (公用按钮配置)
    │   └── user (个人信息)
    ├── views (视图页面)
    │   ├── system (系统管理页面)
    │   └── ...
    ├── router (路由)
    │   ├── routes (公共路由)
    │   └── index.js 路由业务事件处理
